## Общая информация
 * Для написания UI ориентируйся на ручки апишки, описанные в ../src/main/scala/feed/listing/infrastructure/route/ListingRouteImpl.scala - в данном файле представлено описание ручек с помощью Tapir;
 * Пагинация работает по общему механизму: если в запросе указан query-параметр limit, то в ответе указывается заданное количество объявлений, а также поле cursor, данное поле является курсором на следующий батч объявлений, при подгрузке необходимо его передавать в query-параметрах;
