## Ограчниения
 * Текущая директория корневая (директория ./ui в общем проекте), все пути указываются относительно этой директории;
 * Редактировать код вне папки ./ui запрещено;
 * Файлы вне директории ./ui доступны только в режиме READ ONLY;

## Задание
 * Необходимо написать простой ui для работы с api в папке ../src/;
 * ui должен представлять из себя ленту предложений/товаров на подобии авито;
 * Есть реализованные ручки для создания товаров и их просмотра (с пагинацией);
 * Пагинация работает по общему механизму: если в запросе указан limit, то в ответе указывается заданное количество объявлений, а также поле cursor, данное поле является курсором на следующий батч объявлений, при подгрузке, необходимо его передавать в query-параметрах;
 * Ориентируйся на ручки, описанные в ../src/main/scala/feed/listing/infrastructure/route/ListingRouteImpl.scala - в данном файле представлено описание ручек с помощью Tapir;
 * UI должен быть написан на React;
 * UI на текущий момент должен выполнять следующие функциональные требования:
   * Главная страница отображает N количество объявлений в виде плиток (используй авито как референс);
   * Как пользователь пролистает эти N объявлений и дойдет до конца, необходимо асинхронно подгрузить следующие N объявлений (пагинация через поля limit, cursor);
   * При клике на плитку объявления, необходимо открыть страницу с подробной информацией об объявлении;
   * Бэкенд не запущен, проверить ручки нельзя, через ENV-переменную или фича-флаг подготовь фикстурки, чтобы я мог проверить отображение плиток;
   * Авторизации пользователей нет.
